# Baby Movement App

Exercise and milestone tracking app for babies 0-12 months.

## Quick Start

1. **Clone the repo:**
   ```bash
   git clone https://github.com/aparajithn/baby-movement-app.git
   cd baby-movement-app
   npm install
   npm start
   ```

2. **Scan QR code** with Expo Go app on your phone

## Features

- 16 exercises covering 0-12 months (gas relief, tummy time, stretches, coordination)
- 4 quick routines (Gas Relief, Tummy Time, Morning Wake-Up, Bedtime Wind-Down)
- CDC milestone tracker (2, 4, 6, 9, 12 months)
- Built-in timer with progress ring
- Age-based exercise filtering
- Favorite/bookmark system

## Tech Stack

- React Native + Expo
- React Navigation (bottom tabs + native stack)
- AsyncStorage for favorites/completion tracking

## Testing on Your Phone

1. Download **Expo Go** from App Store or Play Store
2. Run `npm start` in the project directory
3. Scan the QR code with:
   - iOS: Camera app
   - Android: Expo Go app
